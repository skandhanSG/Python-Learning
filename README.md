# 🚀 How to Use

Clone the repository

```bash
git clone https://github.com/skandhanSG/Python-Learning.git
```

Move into the project

```bash
cd Python-Learning
```

Run any program

```bash
python filename.py
```

Read the comments and experiment by modifying the code.

The best way to learn Python is by breaking things and fixing them.
